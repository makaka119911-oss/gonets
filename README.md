# Гонец — MVP прототип сервиса доставки

Статический прототип сайта "Гонец" с интерактивным калькулятором и блоком клиентского приложения.

## Что уже сделано

- Лендинг с ключевыми секциями (оффер, услуги, бизнес, FAQ).
- Интерактивный расчет цены (демо-логика на фронтенде).
- Оформление в стиле **Steam** (тёмные панели, голубой акцент #66c0f4, зелёные CTA) и опциональная **светлая тема** (кнопка в шапке).
- Блок "клиентское приложение" в едином стиле с вебом.
- **Логотип** `assets/logo-gonets.svg` и **SVG-иконки** (линейный стиль под тему) в `css/icons.css`.

## Файлы

- `index.html` — основной лендинг.
- `calculate/index.html` — страница калькулятора.
- `tracking/index.html` — страница трекинга заказа.
- `history/index.html` — страница истории заказов.
- `login/index.html` — страница входа.
- `css/styles.css` — дизайн-система и стили.
- `css/icons.css` — размеры и цвет SVG-иконок.
- `assets/logo-gonets.svg` — логотип «Гонец» (Steam-палитра).
- `js/main.js` — интерактивность (меню, тема, калькулятор, демо-форма).
- `js/mock-api.js` — мок-слой API поверх локальных JSON.
- `mocks/*.json` — данные для имитации заказов/трекинга/логина.
- `docs/MVP_SCOPE.md` — зафиксированный объем MVP.
- `docs/ARCHITECTURE_DRAFT.md` — черновик архитектуры.
- `docs/FIGMA_BRIEF_WEB_MOBILE.md` — ТЗ для дизайнера.
- `platform/` — стартовый каркас monorepo Next.js + NestJS.

## Как запустить

Откройте `index.html` в браузере.

## GitHub и онлайн-ссылка

Репозиторий уже можно инициализировать и пушить на GitHub; **автодеплой статического сайта** настроен через GitHub Actions → GitHub Pages. Пошагово: [DEPLOY.md](DEPLOY.md).

## Важно

Это прототип интерфейса. Для production версии нужно:

- подключить реальный backend и БД;
- подключить карты и геокодер (Яндекс/2ГИС);
- подключить платежи, SMS/push, авторизацию;
- реализовать кабинеты клиента/курьера/админа;
- подготовить юрдокументы и политику данных.
